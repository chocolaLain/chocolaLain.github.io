    我们在实数域中继续分划出两个数来，注意数域内此时已经没有空隙了

    基本定理（戴金德）-p12：注意证明开头的一句话，将属于A的一切有理数集记为A，这两个A是不同的集合。
    借助实数域中有理数域来得出分划的实数β，β即为实数域下组A中的最大数，如果不是这样，若A中有比β大的实数α，因为两个实数之内必有有理数（设为r），那么α>r>β，因为r属于实数域A且为有理数，那么r也一定属于实数域A里的有理数域A，此时产生矛盾，因为有理数域下组A是由β分划出来的，不可能再有有理数比β大。

第一章 极限论

整序变量及其极限

主线

  • 了解变量和常量的概念,了解十分一般的变量无法给一个完全形式的定义.
    了解变量和常量如何用集合来表示,例如常量是指X={X},该集合只有一个元素.
  • 建立数列的概念,了解序列,了解序列和整序变量概念上虽然不同,但本质上相同.—p29
    中学学过的算数序列(等差序列)和几何序列(等比序列)遇到的变量都是整序行的变量.
    了解一下内接正多边形的周长公式.
    整序变量的各个值不一定相同.

  • 了解整序变量的极限(即序列的极限定义.) —P31

    含任意ε的不等式(3)就是xn可以与a”相差任意小”这一句话的准确记法,而序号N恰好就只是这上述定义中”从某项开始”那个”某项”的位置
    最重要的就是认识到,序号N一般来说,并不是指定后就永远不变的,它是由我们所选的数ε来决定的,也可以记为Nε

    了解表示极限的点a就是表示整序变量的数值的点的凝聚中心

  • 极限为零的整序变量xn称为无穷小量,简称为无穷小.,也可以将无穷小的概念不用极限来描述而用类似序列极限定义的说法来描述:|xn - 0| = |xn| < ε (n > Nε)
    综上得到命题:整序变量xn以常数a为极限的必要且充分条件是:他们的差αn = xn - a是无穷小.
    得出循环推理:极限由无穷小确定,而无穷小由极限确定
    继续可以得出x = a + αn(无穷小):反之若整序变量满足这表达式,则它有极限a,在实用时常用这个式子以建立变量的极限(例如p34第四题).
  • 例题25 考察了整序变量以及对极限定义的理解 —p33
    • 6) 得出一个推论:整序变量βn = Aq^n |q|<1,而A是常数. *—p35
    • 8)有点难理解,9)的话谈到了无穷级数的概念,这两个需要多看多理解.
  • P37 例26 涉及到有关极限的整序变量的一些定理 例如:若整序变量xn趋于异于零的极限a,则必有充分远的xn的值,其绝对值得超过某正数r: |xn|>r>0 (n>N)
    4和附注涉及了有界变量的证明和解释.
    最后可以利用1`里的例子用反证法证明极限的唯一性
    5讲了整序变量xn不能同时趋于两个相异的极限(极限的唯一性):注意夹在其中间的r,证明极限唯一性或者用唯一性作为推导原理时常用这个r来做参考对比.
  • p38 27 涉及到了无穷大量,注意无穷大量的任何一个数值都不能当做是”大量”.
    了解整序变量xn在称为无穷大并且保持着一定的正负时,可以用正负无穷来表示 例如limxn → 符号(无穷大)/符号(无穷小)

极限的定理·若干容易求得的极限

  • 28 对等式及不等式取极限 —p40
    • 1` 由极限唯一性(可借助26 5’中的r来理解)可推得 由xn = yn得出:lim xn = lim yn
    • 2` 若二整序变量xn，yn常满足不等式xn≥yn，并且各趋于有限极限:limxn=a,limyn=b,则必a≥b,可用反证法来证明
      注意:由严格的不等式xn>yn,仅能推得lim xn ≥ lim yn,不能去掉等号,例如对于一切n,1/n > -1/n,但两者极限都可以趋向于0;
    • 3` 涉及到类似夹逼定理的推论,整序变量xn,yn,zn恒满足不等式 xn≤yn≤zn,且xn以及zn趋向于极限a,则yn的极限也为a.
    • 1’ 2’ 3’都可以推广到无穷极限的情况,但是3’仅适合带确定符号的情况.
