对HTML的认识

  • HTML是用来描述网页的一种语言,HTML指超文本标记语言（Hyper Text Markup Language）,HTML不是编程语言，是一种标记语言
  • HTML.5的新特性：例如用于绘画的canvas标签,用于媒介回放的video和audio元素等
  • 开发工具：常用：WebStorm、notepad++、Eclipse、text sublime、Dreamweaver等,推荐：Intellij IDEA(之后去试试,在html/css上跟PPycharm差不多.)
  • 遇到问题时可以参考的几个解决方法:1、寻求问题根源 2、查看参考文档 3、参考示例 4、常见问题通过搜索引擎搜索 5、问题反馈
  • 需要掌握的技能：HTML 5,XHTML,CSS3,JavaScript,jQuery：jQuery-UI jQuery-Mobile

  • 声明：<！DOCTYPE>
    HTML也有多个不同的版本，只有完全明白页面中使用的确切HTML版本，浏览器才能完全正确地显示出HTML页面。这就是<！DOCTYPE>的用处.HTML5的声明要比之前版本简单一些了.

HTML5元素、属性和格式化-元素

  • 元素:元素指的是从开始标签到结束标签的所有代码(并不指的是元素内容)
    ‘br/‘这是空元素空标签没有元素内容,可以用来换行,为了规范建议加斜杠,与p标签不同的是行间距不同
  • HTML元素语法：
    元素的内容是开始标签与结束标签之间的内容,
    空元素在开始标签中进行关闭,
    大多数HTML元素可拥有属性,
  • 嵌套的HTML元素:大多数的HTML元素都是可以嵌套的(最典型的就是DIV标签)

HTML5链接属性及使用

1、链接数据：
文本链接
图片链接
2、属性：
href属性：指向另一个文档的链接 注意如果链接到一个网址的时候要加上http或者https.
name属性：创建文档内的链接

3、img标签属性：
alt：替换文本属性 例如alt = “Lain的博客”,图片显示不出来时就显示这段文字.
width：宽 例如:width = “100px”
height：高 同上
